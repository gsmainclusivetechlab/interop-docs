    Quick Guide for Tests

    Creating a Session#

    1. Click the New Session Button
    2. Select the System Under Test (SUT) and add its URL
    3. Add the Session Information
    4. Choose the Test Cases for the session
    5. Configure the SUT
    6. Check out the Created Session
    7. Send a Test Request to the Interoperability Test Platform

    Test Results#

    1. Open the Test Session in the Interoperability Test Platform
    2. Access the Test Case of your choice
    3. Check the Test Information
    4. Analyse the Test Run Information to see it pass or fail and why
    5. Check the Test Steps to see all performed tests for each step
    6. Access the Test Flow to see a sequence diagram with the messages between the steps
